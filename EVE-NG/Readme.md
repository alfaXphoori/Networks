# EVE-NG Lab Collection

This repository contains a comprehensive set of hands-on networking labs designed for use with EVE-NG (Emulated Virtual Environment Next Generation). The labs cover fundamental to advanced topics in computer networking, focusing on Cisco IOS, Linux, and multi-vendor interoperability.

## 📚 Lab Directory Structure

- **00_Install/**: EVE-NG installation and setup guides
- **01_Get Start/**: Getting started with EVE-NG and basic navigation
- **02_Add Router Switch/**: Adding routers and switches to your topology
- **03_Switch Config/**: Basic switch configuration labs
- **04_Basic Switch/**: Layer 2 switching fundamentals
- **05_VLAN/**: VLAN configuration and inter-VLAN routing
- **06_Spanning Tree Protocol Lab/**: STP concepts and troubleshooting
- **07_Inter_VLAN/**: Inter-VLAN routing scenarios
- **08_Basic_Routing/**: Static and dynamic routing basics
- **09_RIP_Lab/**: RIP protocol configuration and verification
- **10_OSPF_Lab/**: OSPF single and multi-area labs
- **11_EIGRP/**: EIGRP configuration and troubleshooting
- **12_BGP/**: BGP basics and Internet routing
- **13_ACL_Security/**: Access Control Lists and network security
- **14_NAT_Configuration/**: NAT, PAT, and port forwarding labs

## 🛠️ Requirements
- EVE-NG Community or Professional Edition
- Cisco IOS images (see installation guide)
- (Optional) Linux images for end-host simulation

## 🚀 How to Use
1. Import the provided `.unl` topology files into your EVE-NG environment.
2. Follow the step-by-step instructions in each lab's `Readme.md`.
3. Use the included configuration examples and verification commands.
4. Experiment, break, and fix the labs to deepen your understanding!

## 📖 Recommended Learning Path
1. Start with installation and basic labs (00–04).
2. Progress through VLANs, routing, and switching (05–12).
3. Explore security, NAT, and advanced topics (13–14, Dev).
4. Attempt integrated and exam-style labs in the `Dev/` directory.
